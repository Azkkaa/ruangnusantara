# 🍽️ Rasa Nusantara - Frontend

Frontend aplikasi restoran berbasis React untuk sistem pemesanan makanan
(MVP).

## 🛠️ Tech Stack

-   React + Vite
-   Tailwind CSS
-   Context API (State Management)

## 🎯 Fitur Utama

-   Menampilkan daftar menu (dummy data)
-   Sistem keranjang (multi-order)
-   Checkout (COD)
-   Admin dashboard (lihat & update status pesanan)

## 📂 Struktur Project

    src/
     ├── components/
     ├── pages/
     ├── context/
     ├── data/
     ├── App.jsx
     └── main.jsx

## 🚀 Cara Menjalankan

``` bash
npm install
npm run dev
```

## 📌 Catatan

-   Menggunakan dummy data (belum terhubung backend)
-   State disimpan menggunakan React Context
